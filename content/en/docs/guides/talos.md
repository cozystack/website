## Talos Linux

### Scalable and Reproducible

* Configured with and external manifest, which can be versioned with git and reused. The manifest is declarative, and Talos brings itself to the configured state.
* Whenever you need to find optimal configuration or solve a problem, you do it once and then scale it on any amount of nodes.
* Nodes are guaranteed to have the same configuration and 

### Security

* Protected from user's involvement and configuration drift
* No SSH, removing popular attack vector


### Tailored for Kubernetes

* Kubernetes cluster deployed by Talos
* Full Lifecycle Management for Kubernetes Cluster

### Fine-tuned for Cozystack

* Cozystack provides a fine-tuned Talos build with all required dependencies.
